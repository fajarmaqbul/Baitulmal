Export Asnaf Data dari localStorage

Buka halaman React app Anda (yang sudah ada data Asnaf di localStorage), lalu paste script ini di Console browser:

const asnafData = JSON.parse(localStorage.getItem('asnafData') || '[]');
const rtMap = {'01': 1, '02': 2, '03': 3, '04': 4, '05': 5, '06': 6, '07': 7};

const sqlInserts = asnafData.map(a => {
    const rtId = rtMap[a.rt];
    const nama = a.nama.replace(/'/g, "''");  // Escape quotes for SQL
    return `('${rtId}', '${nama}', '${a.kategori}', ${a.jumlahJiwa}, ${a.tahun}, 'active', NOW(), NOW())`;
}).join(',\n            ');

const sql = `INSERT INTO asnaf (rt_id, nama, kategori, jumlah_jiwa, tahun, status, created_at, updated_at) VALUES
            ${sqlInserts};`;

console.log(sql);
copy(sql);  // Copies to clipboard
alert(`SQL generated! ${asnafData.length} entries ready. Paste dalam file .sql atau via Tinker`);
